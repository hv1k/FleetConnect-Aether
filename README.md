## Quick Start

1. Clone this repo
2. Deploy to [Vercel](https://vercel.com) or any static host
3. Open `fleetconnect/login.html` to get started

## Deployment

### Vercel (Recommended)
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
cd fleetconnect
vercel
```

### GitHub Pages
Push this repo and enable Pages from Settings → Pages → Deploy from branch.

### Any Static Host
Upload the `fleetconnect/` folder to your server. All files are static HTML/CSS/JS.

## Features

- 4 role-based dashboards (Admin, Vendor, Rental Company, Field Worker)
- 49 fully functional pages
- Real-time job tracking & dispatch
- Invoice management & generation
- Equipment tracking & fuel calculator
- Chat messaging system
- Dark/Light mode toggle
- PWA support (installable)
- Responsive mobile design
- Supabase backend integration

## Tech Stack

- Pure HTML / CSS / JavaScript (no build step required)
- Supabase for backend & auth
- Vercel for serverless API routes
- PWA with service worker

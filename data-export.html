API Documentation
GET /api/jobs — List all jobs ▼
Method
GET
URL
/api/jobs?status=pending&limit=50&offset=0
Headers
Authorization: Bearer YOUR_API_KEY
Parameters
• status: pending, assigned, in-progress, completed
• limit: max results (default: 50)
• offset: pagination offset (default: 0)
Example Response
{ "success": true, "data": [{ "id": "job_123", "job_site_name": "Downtown Delivery", "status": "pending", "created_at": "2026-02-08T10:30:00Z" }], "total": 45, "limit": 50 }
